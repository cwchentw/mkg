Using mkgÂ¶

mkg is an opinionated, GNU Make-based project generator that runs on Windows, MacOS, GNU/Linux, and FreeBSD. The objective of this utility is to reduce the routines to write boilerplate Makefiles and to create sensible project structures.

You may find the source of mkg at GitHub or GitLab.
